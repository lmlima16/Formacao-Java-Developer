# Exemplo 1: MinhaClasse
- Aqui aprendi a criar classe a qual inicia-se 99.9% com "public class".- Declarei o nome da classe "MinhaClasse" a qual deverá iniciar com letra maiuscula e se for uma palavra composta a segunda palavra deve-se iniciar também com letra maiúscula.
- abrir chaves duplas para que dentro das mesmas sejam atribuido os métodos a serem utilizados.
- no exemplo da aula utilizamos o método "main" o qual inicia-se com "public static void" main - abrimos e fechamos parenteses para declarardentro destes que utilizaremos "String" a qual deve ser precedida de cochetes[] e o argumento "args" 
- abrimos e fechamos novas chaves para que detro das mesmas sejam iniciado um novo método. Desta vez foi definido o méto de impressão o qual inicia-se com "System.out.println" na sequencia abre-se e fecha-se parenteses para que dentro destes sejam declarado, desta vez entre aspas duplas o que vai ser impresso. Imprimimos como exemplo "Olá Mundo!"
- finalizamos revisando chaves, identação, e escritas.
